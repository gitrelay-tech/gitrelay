# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/) and this project adheres to
[Semantic Versioning](https://semver.org/).

## [Unreleased]

### Added
- Public Prompt Library with search and sort
- Deep Reverse and Manual control modes
- Provider-agnostic LLM router (`auto`, grok, openrouter, openai, google)

- GitHub community files, CI workflow, and architecture docs

## [0.1.0] - 2026-06-01

### Added
- Initial release: reverse-engineer a GitHub repository into a build-ready prompt
- Quick reverse via `POST /api/reverse-prompt`
- Premium pricing page and Stripe checkout scaffolding
- Optional Supabase caching layer
